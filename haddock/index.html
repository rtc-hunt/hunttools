hunttools-0.1.0.0

A collection of tools for doing mystery hunt puzzles.

Modules

  • Anagram Dictionary structure and queries for anagrams.
  • AnswerChecker Check answers for puzzlehunt 2016 - Monsters et Manus - without seeing them.
  • Cipher Some very basic cipher tools.
  • ConsumptiveMonad a monad for query-consuming operations, like multi-word anagrams or word deinterlacing.
  • Crossword In-order regex-like dictionary searches.
  • DBPedia SPARQL queries against DBPedia, a structured extract of Wikipedia.
  • Dictionary Select the correct preprocessed form of a dictionary automatically based on the query type
  • Elements Basic chemical element information.
  • MergeDawg Tools for building larger DAWGs for dictionaries. Mostly needed for handling onelook-sized data.
  • Pipe Infix reversed-composition operator. $| is analogous to shell pipes.
  • Scrabble Scrabble point scores.
